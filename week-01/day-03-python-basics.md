## Python basics

- learned how `print()` works and how it displays values in the terminal
- understood that `print()` is often used for debugging during development
- learned how `input()` pauses program execution and allows the user to enter data
- experimented with `sep` and `end` arguments in `print()` to control output formatting

## Variable types

- learned basic variable types in Python
- `int` — integer numbers
- `str` — string values

## Numeric types

- Integer — whole numbers
- Float — numbers with a decimal point

## Integer operations

- `**` exponentiation
- `//` integer division
- `%` remainder of division

## Naming conventions in Python

- `snake_case` — variables, functions, methods, modules
- `CamelCase` — classes
- `kebab-case` — package names
- `CONSTANT` — constants
